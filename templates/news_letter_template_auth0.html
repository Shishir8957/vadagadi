Auth0 Guardian
Guardian makes adding multifactor authentication easy for end users. No more copying one-time codes - just download the app and approve login requests from your smartphone. Turn it on with one click in your Auth0 dashboard.
Breached Password Detection
We’ve all re-used a password before. It’s easy to do, but unfortunately not a safe practice, as leaked credentials on 3rd party sites like Dropbox, LinkedIn, OneLogin, Home Depot, Target, Anthem Health, and others we see in headlines showcase. Stop hackers trying to use these stolen passwords to break in by protecting your users with Breached Password Detection. In one click you can turn on the ability for users to be notified if the signature of a password they’re using has been previously compromised on a 3rd party site.
More of the Good Stuff
Password Policy
We’ve added password policy support for Password Dictionary (prevents the use of common passwords) and Password Personal Data (prevents the use of personal data in a password). You can choose to enable either or both through your Auth0 dashboard by clicking on “password policy”.


Github and Bitbucket Deployment Extensions
Deploy Rules and database connection scripts from Bitbucket to Auth0 or from GitHub to Auth0 with the Bitbucket and GitHub deployment extensions. You can find them in your Auth0 dashboard under “Extensions”.


Custom Social Connections Extension
Want to add a social connection not on the default list? The Custom Social Connections Extension makes it easy to add any identity provider that supports OAuth 1, OAuth 2 or OpenID Connect. Check out the documentation here and find the extension in your Auth0 dashboard (or contact us if you’d like to retain our professional services team to create the connection for you).
